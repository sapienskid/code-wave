# Infinity CodeWave

Nepal's Premier Data & Computing Hackathon

A 48-hour hackathon where data and mathematics meets code.

## Tech Stack

- Next.js 14
- React 18
- Tailwind CSS
- Lucide React

## Installation

```bash
git clone https://github.com/yourusername/code-wave.git
cd code-wave
npm install
npm run dev
```

Visit `http://localhost:3000`

## Configuration

Edit `src/data/config.js` to customize:
- Event details
- Tracks
- Sponsors
- FAQs
- Social links

## Deployment

```bash
npm run export
```

Deploy the `out/` directory to Cloudflare Pages, Vercel, or Netlify.

See [CLOUDFLARE_DEPLOYMENT.md](CLOUDFLARE_DEPLOYMENT.md) for details.
